# Contact Management System

A simple contact management system that combines:
- **Doubly Linked List** for forward/backward traversal
- **Hash Table** (JavaScript object) for O(1) lookups by exact name
- **Naive String Matching** for substring search

## Features

- Add contacts with name and phone number
- Search by keyword (substring match using naive algorithm)
- Search by exact name (hash table lookup)
- Display all contacts in forward order
- Display all contacts in backward order

## Data Structures Used

| Structure          | Purpose |
|--------------------|---------|
| Doubly Linked List | Store all contacts; allow forward/backward display |
| Hash Table (Object)| Fast lookup by exact name |
| Naive String Match | Substring search for keyword matching |

## How to Run

1. Save the code as `index.html`
2. Open the file in any modern web browser
3. Use the buttons and input fields to manage contacts
